# Data-analysis-Disease-Tolerance
This repository uses data from the publication: Disease tolerance and infection pathogenesis age-related tradeoffs in mice by Sanchez et al. (Nature, 2026). Available at: https://www.nature.com/articles/s41586-025-09923-x#Sec9
1. RNA-Seq_Heatmap recreates the heatmap to compare gene expression of few selected genes
2. KEGG pathway analysis for young and aged mice replicates the KEGG pathway analysis done in the paper to recreate the results for enrichment in the FoxO signaling pathway

GSE178878_raw.txt.gz is available at https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE178878
